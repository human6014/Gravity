# Gravity
혼자서 만드는 1인칭 서바이벌 게임

조작 키

	W,A,S,D                 이동
	Ctrl                    웅크리
	Space                   점프
	Shift                   달리기
	1 ~ 5                   무기 변경
	Q                       플래시 라이트
	E                       힐 킷 사용
	N                       사격 모드 변경 (원거리 무기만)
	Z,X,C                   변경할 중력 방향 선택 각각 X,Y,Z
	마우스 스크롤 위, 아래    Z,X,C에서 선택된 축으로 위,아래 중력 변경
	ESC                     일시 정지
